Mail2
=====

This is a simple tool to send emails from command with an SMTP server. Thus
you could type:

    $ mail2 -smtp="smtp.orange.fr:25" -from="michel.casabianca@gmail.com" \
      -to="michel.casabianca@gmail.com" -subject="Test" -attach="mail2.go" \
      "This is a test!"

This is that simple!

*Enjoy!*
